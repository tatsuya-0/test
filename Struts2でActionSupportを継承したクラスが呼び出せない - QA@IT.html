Struts2でActionSupportを継承したクラスが呼び出せない

こんにちわ。
初めての投稿で、至らない点があるかもしれませんが、
どうぞご容赦下さい。

以下の記事を参考に、Struts2の学習を行っています。
http://www.atmarkit.co.jp/fjava/special/struts2/struts2_1.html

上記記事と同様に、以下のファイルを作成しました。

・index.jsp
・index-hello.jsp
・indexAction.java
・web.xml

Tomcatを起動し、コンテキストの起動は成功します。
その後、index.actionにアクセスすると、index.jspの内容が表示されるます。
ここまでは問題ないのですが、ここで「こちらこそこんにちわ」ボタンを
押すと、以下のエラーが発生します。

java.lang.NoSuchMethodException: com.opensymphony.xwork2.ActionSupport.hello()

エラーの内容から察するに、ActionSupportにhelloなんてメソッドはないよってことだと思うのですが、
何故ここでindexActionではなく、継承元のActionSupportが呼ばれてしまうのでしょうか？

また、本件と関係あるかわかりませんが、Tomcatに以下の警告が表示されています。
警告: No configuration found for the specified action: 'index' in namespace: '/'. Form action defaulting to 'action' attribute's literal value.

以下、スペック等について記載します。

・Windows XP Professional SP3
・Tomcat 5.5.27
・Eclipse 3.4.2
・Struts 2.1.6
・JDK1.6.0.13

以上、長くなりましたがよろしくお願い致します。

はい。間違いなくtestパッケージになっております。

以下、私の認識が間違っているかもしれないのですが、
http://localhost:8080/sample/index.action
にアクセスした場合、indexAction.javaのexecuteメソッドが呼ばれ、
その結果がindex.jspで表示されると思っていましたが、
本当にindexAction.javaのexecuteが実行されているのかを
確認するため、index.jspおよびindexAction.javaのソースを
以下のように変更しました。

【indexAction.java】
package test;

import java.util.Map;

import com.opensymphony.xwork2.ActionSupport;

public class indexAction extends ActionSupport{

Map&lt;String, String&gt; sessionMap;

public Map&lt;String, String&gt; getSessionMap() { return this.sessionMap; }
public void setSessionMap(Map&lt;String, String&gt; s) { this.sessionMap = s; }

public String execute() throws Exception {
    sessionMap.put(&quot;test&quot;,&quot;AAA&quot;);
    return &quot;success&quot;;
}

public String message;

public String hello() throws Exception {
    message = &quot;いえいえこちらこそこんにちは。&quot;;
    return &quot;hello&quot;;
}

}

【index.jsp】
<%@ page language="java" contentType="text/html; charset=Windows-31J" pageEncoding="Windows-31J"%>
<%@ taglib prefix="s" uri="/struts-tags" %>
<html>
<body>
<s:form>
こんにちは。<s:property value="#sessionMap.test" />
<s:submit method="hello" value="こちらこそこんにちは"/>
</s:form>
</body>
</html>

ここで私の期待する結果としては、index.jspの実行結果にて、
「こんにちわ」の後に「AAA」が表示されるものだと思っていましたが、
表示されたのは「こんにちわ」だけでした・・・
